## 컨셉

---

> **위기에 빠진 친구를 구하라!**
> 

이런, 당신의 친구 버섯이 이 땡볕에 인간 마을 지붕에 갇혀버렸군요!

버섯들의 영웅, 용감한 당신이 나서야겠군요? 인간들이 만들어놓은 구조물들을 이용하여, 지붕에 도달하여 친구를 구해봅시다.
